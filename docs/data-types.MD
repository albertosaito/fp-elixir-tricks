# Elixir basic types

``` elixir
iex> 1                         # integer
iex> 0x1F                      # integer (hexadecimal)
iex> 1.0                       # float, because the dot
iex> true                       # boolean
iex> :atom                      # atom
iex> "elixir"                  # string (remember they're always utf-8 by default "hellö wörld 由力驱动")
iex> [1,2,3]                   # list
iex> {1,2,3}                   # tuple
```
